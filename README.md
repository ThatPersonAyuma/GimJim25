Global Script 
Gunakan class "Global" untuk akses:
- McHealth (Gunakan Global.take_damage untuk mengurangi health pemain)
- MovePush (Mempengaruhi pergerakan player secara permanen, reset jadi Vector(0,0) jika sudah)
- McKnockBack (Untuk memberi efek knockback ke pemain)
- char_stun (Efek stun)

Gunakan Contoh Script enemy pada res://StageJantung/boss_jantung.gd
